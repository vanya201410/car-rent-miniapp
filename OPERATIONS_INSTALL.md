# Фотоосмотр, залог, штрафы и черный список

Что добавлено:

1. **Фото до выдачи**
   - кнопка в админке Telegram: `📸 Фото до`
   - после нажатия отправляй фото автомобиля в Telegram по одному
   - когда закончишь — напиши `/done`

2. **Фото после возврата**
   - кнопка в админке Telegram: `📸 Фото после`
   - загрузка работает так же: фото по одному → `/done`

3. **Управление залогом**
   - `💰 Залог получен`
   - `✅ Залог возвращен`
   - `⚠️ Удержать залог` — бот спросит сумму и причину

4. **Штрафы / платные дороги**
   - кнопка `🚦 Штраф`
   - бот спросит сумму и причину
   - к штрафу автоматически фиксируется админ-сбор 25 €

5. **Возврат автомобиля**
   - `✅ Возврат без проблем`
   - `🧽 Нужна мойка`
   - `⛽ Не хватает топлива`
   - `🕒 Опоздание`
   - `🏁 Завершить аренду`

6. **Черный список**
   - кнопка `⛔ В черный список`
   - клиент сохраняется в таблицу `blocked_clients`

## Что сделать после загрузки архива

1. Залей файлы в GitHub.
2. В Supabase снова запусти SQL:

```text
supabase/extra-services-setup.sql
```

3. Дождись деплоя в Vercel.
4. Открой Telegram админку: `/admin`.
5. Перейди в `✅ Подтверждённые` и проверь новые кнопки.

## Важная логика

Фотоосмотр лучше делать так:

Перед выдачей:
- перед авто
- зад авто
- левый бок
- правый бок
- салон
- приборная панель с пробегом
- уровень топлива
- колеса / диски

После возврата — то же самое.

Это защищает залог и помогает решать спорные ситуации.
